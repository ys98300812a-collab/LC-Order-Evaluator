# 📦 電商智能叫貨評估系統 (Order Evaluator)

這是一個專為電商營運與實體通路打造的「單頁式自動化叫貨評估工具」。透過前端運算直接解析平台歷史訂單，一鍵自動清洗混亂料號，並產出包含安全庫存建議的終極採購報表。

## ✨ 核心價值與功能

* **📊 動態安全庫存分析**：支援自訂緩衝比例（如預設 +30%），即時試算各通路採購需求，精準控管庫存資金。
* **🤖 智能料號清洗引擎**：內建防呆機制，自動將混雜尺寸、顏色、新舊版的歷史 SKU（如：Yaber 投影布幕、LC 全自動垃圾桶等）強力對齊至標準料號。
* **🏢 通路自動拆分**：自動辨識並拆分「經銷商」與「實體店面」的需求量，生成獨立作業表單。
* **⚡ 一鍵匯出終極報表**：自動生成包含「多品牌分類」、「通路明細」及「老闆備查專用（客戶下單明細）」的完美 Excel 採購單。
* **🔒 高度隱私安全**：純前端 (Browser-side) 運算架構，無伺服器資料傳輸，確保公司營業數據絕對安全，不外洩。

## 🚀 快速開始

1. 準備好平台匯出的「3 個月歷史訂單 CSV」與「客戶通路對照表」。
2. 開啟本系統網頁連結。
3. 拖曳上傳檔案，並視當月檔期需求設定「安全庫存百分比」。
4. 點擊運算，一秒下載整理好的多頁籤 Excel 採購報表。
