# 📦 電商智能叫貨評估系統 (Order Evaluator) - Pro v1.3

這是一個專為電商營運與實體通路打造的「單頁式自動化叫貨評估 SaaS 工具」。透過純前端運算技術，解析平台歷史訂單並自動清洗混亂料號，產出具備科學建議的終極採購報表，徹底終結傳統 Excel 人工對帳的耗時痛點。

## 🚀 核心價值 (Core Values)

* **📊 動態安全庫存分析**：支援自訂緩衝比例（預設 +30%），精準控管庫存水位，優化資金利用率。
* **🤖 智能料號清洗引擎**：自動將混雜尺寸、顏色、新舊版的 SKU 強力對齊至標準料號，確保數據統計準確。
* **🏢 通路自動拆分**：自動辨識並拆分「經銷商」與「實體店面」的需求量，實現精細化作業。
* **⚡ 一鍵匯出終極報表**：自動生成包含「多品牌分類」、「通路明細」及「老闆備查專用（客戶下單明細）」的完美 Excel。
* **🔒 高度隱私安全**：100% 純前端 (Browser-side) 運算，營業數據不經過任何伺服器，保障企業機密。

## 📖 快速開始 (Quick Start)

1.  準備平台匯出的「3 個月歷史訂單」與「客戶通路對照表」。
2.  開啟系統網頁，並拖曳上傳上述檔案。
3.  設定當月檔期所需的「安全庫存百分比」。
4.  點擊運算，一秒下載整理好的多頁籤採購建議表。
